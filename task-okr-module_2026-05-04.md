# Life OS OKR 模块 - 2026-05-04

## 目标
为 Life OS 项目添加 OKR（目标与关键结果）管理模块

## 创建的文件

### 1. src/views/OKR.vue
完整的目标管理组件，含：

**顶部统计**：目标总数 / 已完成 / 进行中 / 整体进度（4张卡片）

**Objective 管理**：
- 新增/编辑/删除弹窗（标题、描述、季度、状态）
- 状态标签：进行中（蓝）、已完成（绿）、暂停（灰）
- Objective 进度 = 所有 KR 进度的平均值
- 进度条颜色：红(<20%) → 黄(20-40%) → 蓝(40-80%) → 绿(80%+)

**Key Results 管理**：
- 每条 KR 有 SVG 圆环进度指示器
- 进度滑块拖动调节当前值（实时保存）
- 显示格式：当前值 / 目标值 单位
- 悬停显示编辑按钮

**季度筛选**：下拉选择季度过滤，自动包含当前+下季度

**持久化**：数据保存到 localStorage

### 2. public/data/okr.json
4 个 Objective，各含 3 个 KR，覆盖教学质量/科研/个人成长/公众号四领域

### 3. src/main.js（修改）
添加 OKR 路由 `/okr`
